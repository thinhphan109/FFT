1. Đặt vấn đề

FFT là gì?

FFT (Fast Fourier Transform) là thuật toán biến đổi tín hiệu từ miền thời gian sang miền tần số.

Giảm độ phức tạp DFT từ O(N²) xuống O(N·logN).

Ứng dụng

  • 📡 OFDM trong 4G/5G/Wi-Fi
  • 🎵 Nén âm thanh MP3, AAC
  • 🖼️ Nén ảnh JPEG (DCT họ FFT)
  • 📊 Phân tích phổ, đo dao động
  • 🛰️ Radar, sonar

3. Lý thuyết DFT

Công thức Discrete Fourier Transform

X[k] = Σ x[n] · e^(-j·2π·k·n/N)        n = 0 .. N-1

Với N = 8, cần 64 phép nhân phức và 56 phép cộng phức ⇒ O(N²).

Thuật toán FFT (Cooley–Tukey 1965)

Phân chia DFT N điểm thành 2 DFT N/2 điểm theo:

  • DIT — Decimation In Time (chia theo n chẵn/lẻ)
  • DIF — Decimation In Frequency (chia theo k thấp/cao)

Đồ án này sử dụng Radix-2 DIT.

5. Định dạng Q1.15

Format

Bit  : 15 14 13 ... 0
       ┌──┬─────────────┐
       │S │ 15 frac bits│
       └──┴─────────────┘
Range: [-1.0, +1.0)
LSB  : 1/32768 ≈ 30.5 μ

Quy tắc nhân

Q1.15 × Q1.15 = Q2.30
truncate bits [30:15] → Q1.15

A = 0.5  → 16384
B = 0.5  → 16384
A × B = 0.25
       = 0x10000000 (Q2.30)
       → 0x2000 = 8192 (Q1.15)

7. Butterfly Radix-2 DIT

Phương trình

y0 = (A + B·W) / 2
y1 = (A − B·W) / 2

scale 1/2 mỗi stage
→ tổng scale 1/N = 1/8
→ chống overflow Q1.15

Sơ đồ

A ──────●──[+]── y0
        │   │
        │  [-]── y1
        │   │
B ─[×W]─●───┘

8. Mã nguồn — complex_mul

module complex_mul #(parameter W = 16)(
    input  signed [W-1:0] a_re, a_im, b_re, b_im,
    output signed [W-1:0] y_re, y_im
);
  wire signed [2*W-1:0] m_rr = a_re * b_re;
  wire signed [2*W-1:0] m_ii = a_im * b_im;
  wire signed [2*W-1:0] m_ri = a_re * b_im;
  wire signed [2*W-1:0] m_ir = a_im * b_re;

  wire signed [2*W-1:0] real_full = m_rr - m_ii;
  wire signed [2*W-1:0] imag_full = m_ri + m_ir;

  assign y_re = real_full[2*W-2 : W-1];   // Q2.30 → Q1.15
  assign y_im = imag_full[2*W-2 : W-1];
endmodule
